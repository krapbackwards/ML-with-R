Ensemble Models

  • These models are created by integrating two or more weak or strong base learners with the objective of generating a final robust model that provides a classification or prediction by incorporating the information from all base learners.
  • This methodology alleviates the bias observed for any individual model.
  • Additional to Ensemble method implementation it is important to tune parameters(Example: Number of trees),choose the metric(Example:Accuracy) and the test algorithm (Example:Random Forest) to find the optimal global maxima. It is also paramount to use techniques like crossvalidation to stop overfitting of the data.

Types of Ensemble Models

  1. Averaging
  • The final prediction is obtained by averaging the values of the models like Regression as well as averaging of probabilities.

Ensemble Implementation Techniques

  1. Bagging

  • Bagging or Bootstrap aggregation implements boot strap sampling to generate training samples.

  • The bootstrap samples are for each ML learner is created by sampling with replacement for each learner.

  • The samples are individually run through a particular ML algorithms and finally the model classification is decided by the majority vote or averaging for the objective of prediction.

  • This technique is very beneficial for scenarios where the algorithm is unstable in context to small changes in the input data.

  • Trees have a propensity of being very sensitive to the input data therefore ensemble can be effectively used in this context.

  • Random Forest in conjunction with implementing the bootstrap scheme selects a random number of features used as split criterion. This startegy helps further decrease the variance.

  • Ipred and caret package provides the bagging implementation.

  1. Boosting
  • Boosting like bagging generates resampled samples but it is used to create complementary learners (can be more than two)and higher voting weightage is attributed to the learners that a better prior performance.
  • This algorithm entails training the first model using the entire data and then directing the residual from the first learner progressively into the the other learners in order to improve prediction incrementally. Every step gives more weightage to observations that were poorly predicted.
  • This algorithm is focused on alleviating the bias within the system though to avoid overfitting the data parameter tuning is important.
  • The examples of Boosting is XBoost,GBM,Adaboost.
  • This mechanism ensures that weak learners perform as well as the strong learner if not better.
  • AdaBoost is a tree based Boosting technique.
  • This is considered to be a very innovative and novel discovery in contemporary times.

Bagging : Bootstrap Aggregation Ensemble

3.Stacking

  • This ensemble technique implements a layered approach where the intial level has a set of algorithms that generate predictions that are fed into the next level of algorithms that improve these and provide the final predictions.The final level can also use other methods like Averaging,Majority Voting or Weighted Averaging.

  • The considerations for choosing models used within the same level are a prediction accuracy threshold and the disparate model predictions are uncorrelated.
